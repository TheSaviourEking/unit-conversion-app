
# Unit Converter

A simple unit converter with dynamic functions

Can be used to convert:
### Temperature: 
from Celsius(c) to Fahrenheit(f) and vice versa

### Weight:
from Kilograms(KG) to Pounds(p) vice versa

### Distance:
from Kilometers(KM) to Miles(m) vice versa
